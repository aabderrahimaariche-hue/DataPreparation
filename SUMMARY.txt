MODULE 1: GROUP INTELLIGENCE (10_group_intelligence.R)


 compute_identifier_statistics()
    Aggregates ANY identifier by ANY numeric value
    Outputs: mean, std, min, max, median, count, variance
    Use Cases: User stats, Item stats, Customer stats, Sensor stats
    Example: compute_identifier_statistics(data, "userId", "rating")
    Example: compute_identifier_statistics(data, "customerId", "purchase")

 compute_identifier_deviation()
    Calculates bias/deviation from reference value
    Outputs: absolute deviation, percentage deviation
    Use Cases: User bias detection, Customer deviation, Item preference
    Example: compute_identifier_deviation(data, "userId", "rating", "movie_avg")

 classify_identifier_patterns()
    Converts statistics into categorical patterns
    Patterns: valeurization_level (high/medium/low), consistency (consistent/medium/variable)
    Use Cases: Rater classification, User profiling, Behavior patterns
    Example: classify_identifier_patterns(stats, "rating", "consistency")


MODULE 2: CATEGORICAL INTELLIGENCE (11_categorical_intelligence.R)


  encode_categorical_flags()
    One-hot encoding for PIPE|DELIMITED|CATEGORIES
    Works with ANY delimiter: |, comma, semicolon, etc.
    Outputs: Binary columns (0/1) for each category
    Examples: genres, tags, symptoms, attributes, keywords
    Example: encode_categorical_flags(data, "genres", "movieId", "\\|")

 count_categories()
    Counts categories per entity + multi-category flag
    Outputs: category_count, is_multi_category
    Use Cases: Diversity analysis, Specialization detection
    Example: count_categories(data, "movieId", "genres", "\\|")

 extract_primary_category()
    Gets first category from delimited string
    Outputs: primary_category column
    Use Cases: Main genre, primary tag, dominant symptom
    Example: extract_primary_category(data, "movieId", "genres", "\\|")

 compute_category_statistics()
    Aggregates per category: mean, std, count, median
    Works with ANY categorical + ANY value combination
    Outputs: per-category statistics
    Examples: Rating by genre, Price by tag, Outcome by diagnosis
    Example: compute_category_statistics(data, "genres", "rating", "\\|")

  compute_entity_category_preferences()
    User/Customer/Entity preferences for each category
    Calculates: deviation from global mean per entity-category pair
    Outputs: Pivot format (entity_id + pref_category1 + pref_category2...)
    Examples: User genre preferences, Customer product preferences
    Example: compute_entity_category_preferences(data, "userId", "genres", "rating", "\\|")

 compute_category_diversity()
    Measures niche vs mainstream categories
    Outputs: category_diversity, is_niche, is_mainstream flags
    Use Cases: Market analysis, Content classification
    Example: compute_category_diversity(data, "genres", "\\|")

 compute_category_combinations()
    Finds common multi-category patterns
    Outputs: combination frequency ranking
    Examples: Genre pairs, Tag combinations, Attribute pairs
    Example: compute_category_combinations(data, "movieId", "genres", "\\|")
